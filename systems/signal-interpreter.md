# Signal Interpreter

The Signal Interpreter converts user input into emotional and symbolic discovery patterns.

## Input Types
- words
- symbols
- moods
- random typing
- phrases
- aesthetics

## Interpretation Goals
Detect:
- emotional tone
- visual atmosphere
- rhythm
- symbolic meaning
- discovery direction

## Example

Input:
"moon drift static"

Possible interpretation:
- nighttime aesthetics
- ambient music
- dreamlike visuals
- soft blue environments
- emotional exploration

## Core Philosophy

Chaos is not treated as error.

Chaos is treated as compressed signal.
