# Web Discovery

The engine helps users discover pictures, websites, videos, music, and communities through signal-based searching.

## What It Can Find
- images
- websites
- videos
- music
- communities
- aesthetics
- ideas
- worlds

## How It Works
The user types words, moods, symbols, or random signal.

The engine translates that input into discovery paths.

## Example

Input:
"cute moon garden"

Discovery results:
- cozy moon artwork
- garden websites
- calm music
- cute fantasy images
- soft nighttime aesthetics

## Goal

Make searching the internet feel alive, creative, and personal.
